# A/B Test Conversion Rate Dashboard

Простой дашборд для визуализации коэффициента конверсии (CR) по A/B-тестам.

## Возможности

- Автоматический расчёт CR: `(conversions / visits) * 100%`
- Сравнение до 4 вариаций (Original, Variation A, B, C)
- Интерактивный график с вертикальной линией при наведении
- Tooltip с сортировкой: **самые высокие проценты — сверху**
- Управление отображением вариаций через чекбоксы (всегда выбран хотя бы один)
- Адаптивный дизайн (поддержка экранов от 671 до 1300+ пикселей)
- Все значения отображаются в процентах
- Смена темы

## Как запустить

   ```bash
   npm install

   npm run dev
   ```
